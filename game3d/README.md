# game3d

## Responsabilite
Contenir le projet Godot 3D (scenes, scripts, data) du sandbox.

## Contenu principal
- `project.godot`: point d'entree du projet.
- `scenes/`: scenes Godot.
- `scripts/`: logique GDScript par domaine.
- `data/`: donnees JSON de sandbox.

## Notes
- Le dossier `.godot/` est genere par l'editeur et n'est pas documente dossier par dossier.
