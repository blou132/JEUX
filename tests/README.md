# tests

## Responsabilite
Regrouper les tests unitaires et de non-regression du depot.

## Contenu principal
- Fichiers `test_*.py` couvrant gameplay, exports, CI support metrics et contrats de sortie.
- `fixtures/`: jeux de donnees de test.

## Notes
- Commande standard: `py -m unittest discover -s tests -p "test_*.py"`.
